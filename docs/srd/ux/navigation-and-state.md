# 用户体验 — 导航和状态

## 1. 返回导航状态保留
- 从市场详情页返回时，“返回体育主页”必须保留：
  - 选定的体育类别筛选条件
  - 滚动位置（推荐；策略待定）

## 2. 入口点
- 用户可以通过以下方式进入市场详情页：
  - 点击市场卡片主体
  - 点击选项预览（快速投注意图）

## 2.1 登录门控（MVP）
- 当用户尝试执行需要认证的动作（例如下注、创建市场）但当前未登录时：
  - 系统必须打开登录对话框/登录流程入口点；
  - 返回后应恢复用户的上下文（例如仍停留在当前市场详情页；不应无故丢失已输入的下注金额）。

## 3. 模态行为（市场创建）
- “创建市场”是一个全屏模态；关闭它不应丢失底层体育主页的筛选状态。

## 4. 侧边栏筛选（MVP）
- 联赛筛选应支持“再次点击取消选择”（回到 `all`/未筛选状态）。
