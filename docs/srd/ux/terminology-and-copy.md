# 用户体验 — 术语和文案

## 1. 标签标准
- 市场状态标签：
  - ACTIVE（活跃）： “进行中”
  - CLOSED（关闭）： “已结束”
  - SETTLED（已结算）： “已结算”
- “我的持仓”部分：包含说明“市场结算后，持仓将自动计算。”
- 回报/收益口径（必须在所有投注入口保持一致）：
  - “潜在回报”：若所选选项获胜，用户可能收到的**总金额**，计算为 `本金 * oddsDecimal`。
  - “净利润”：`潜在回报 - 本金 - 费用`。

## 1.1 登录与行动入口（MVP）
- 未登录下注入口建议文案：“登录以进行投注”

## 2. 负责任交易提示
- 趋势标签页应包含一致的提示信息，强调负责任的参与和风险意识。

## 3. 错误文案指南
- 倾向于使用简短、具体的提示信息：
  - “余额不足。请充值后重试。”
  - “余额不足”（按钮/提示态）
  - “立即充值”（余额不足后的行动入口）
  - “市场已结束。无法下注。”
  - “网络错误。请重试。”
