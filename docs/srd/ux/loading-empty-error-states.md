# UX — 加载、空和错误状态

## 1. 加载
- 首页：加载时显示骨架屏市场卡片。
- 详情页：显示加载指示器；如果部分区域可以渲染，则不应阻止整个视图。
- 提交操作：
  - 投注确认按钮显示加载状态；防止重复点击。
  - 市场创建提交显示全屏加载蒙版。

## 2. 空状态
- 没有活跃事件：显示插图 + “没有活跃事件，请稍后再查看”。
- 筛选无结果：显示“此类别中没有事件”。
- 没有持仓：显示“您还没有持仓；请选择一个选项开始”。

## 3. 错误状态
- 网络错误：Toast 提示 + 重试操作。
- 余额不足：模态对话框 + 充值导航。
- 市场已关闭：Toast 提示“市场已关闭；无法投注”。
- 投注失败：带有具体原因的 Toast 提示；保留用户输入。
