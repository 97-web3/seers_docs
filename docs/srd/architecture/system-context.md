# 系统背景

## 1. 参与者
- **最终用户**: 浏览市场、查看趋势、下注、监控仓位。
- **博彩公司（创建者）**: 在博彩公司模式下配置和创建市场。
- **运营人员**: 为客户端体验配置特色/置顶市场和系统配置。

## 2. 外部系统
- **体育数据提供商**: 提供体育类别、联赛、赛事、队伍、开始时间和状态更新。
- **钱包/支付**: 提供余额、扣款和充值入口。
- **媒体存储**: 存储和提供市场封面图片（预设库和上传的）。
- **管理/运营提供商**: 运营人员配置（特色/置顶、玩法配置、图片预设）的来源。
- **结算/裁决提供商**: 裁决决定和结算结果的来源（可能是内部或外部的）。

## 4. 系统边界
- SRD 规定了**客户端行为和所需的服务器接口**，以支持体育预测市场体验。
- 体育赛事定义和官方结果是**外部输入**；市场结算基于裁决决定。
- 财务余额和扣款是**外部或独立拥有的**能力，通过明确的合同集成。
