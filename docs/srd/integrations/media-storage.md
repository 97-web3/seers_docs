# 集成 — 媒体存储

## 1. 目的
通过预设目录和创作者上传来支持市场封面图片。

## 2. 要求
- 预设目录：
  - 必须提供市场封面图片的预设列表。
  - 预设必须具有稳定的标识符和 URL。
- 上传：
  - 必须支持上传最大 2MB 的 JPG/PNG（策略）。
  - 推荐尺寸：800×450；除非策略要求严格验证，否则执行可能仅为警告。
  - 响应必须返回稳定的 `mediaId` 和可检索的 `url`。

## 3. 接口要求
有关端点和语义，请参阅 `api-specifications/market-creation.md`。

## 4. 失败模式和回退
- 如果预设检索失败：允许使用默认封面创建（如果允许）或使用清晰的消息阻止（策略决定）。
- 如果上传失败：允许重试并保留其他表单数据。
