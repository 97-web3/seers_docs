# 1. 产品概述

# 1.1 产品定位

体育预测市场平台是一个基于预测市场机制的体育赛事竞猜平台，用户可以通过购买不同结果的份额来表达对赛事结果的预测。平台采用市场化定价机制，价格实时反映市场对各结果发生概率的预期。

# 1.2 核心功能

- 赛事浏览：查看热门赛事、按体育类别筛选赛事

- 预测参与：选择预测选项、下注投资、实时查看赔率变化

市场创建：用户可自主创建预测市场（做庄模式）

- 持仓管理：查看我的持仓、潜在收益、对冲提示

# 2.1 页面结构

首页分为两大区域：热门赛事预测区和全部赛事列表区。

# 2.1.1 热门赛事预测区

- 标题："热门赛事预测"

- 展示规则：展示当前最热门的 3 场赛事（以 UI 设计的最终版为准）

- 卡片布局：横向排列 3 张赛事卡片，支持左右滑动（移动端）

- 推荐位选取逻辑：

- 优先级 1：运营手动置顶的赛事，通过后台热门赛事配置

优先级 2：投注金额最高的进行中赛事

优先级 3：参与人数最多的进行中赛事

# 2.1.2 全部赛事列表区

- 标题："全部体育 X 场比赛" (X 为当前筛选结果数量)

- 左侧边栏：体育分类筛选器

- 右侧主区域：赛事卡片网格（3 列布局）

# 排序优先级（降序排列）：

优先级 1：置顶权重（运营后台配置）

优先级 2：热度分数 = 参与人数 $\times 0.4 +$ 投注金额(标准化) $\times 0.4 +$ 近 24 小时增量 $\times 0.2$

优先级 3：创建时间（新创建的优先）

# 7.2 收益计算公式

# 7.2.1 潜在回报计算

潜在回报 = 下注金额 $\times$ (1 / 当前胜率)

示例:下注$50 购买 48%占比的选项,潜在回报=$50×(1/0.48)=$104.17

# 7.2.2 手续费计算

手续费 $=$ 下注金额 $\times$ 手续费率（当前为 $0.2\%$ ）

示例:下注$50,手续费=$50×0.2%=$0.10

# 7.2.3 净收益计算

净收益 $=$ 潜在回报-下注金额-手续费
